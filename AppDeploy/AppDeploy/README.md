# AppDeploy

AppDeploy started as an idea to make testing HTML apps as fast as possible — no commands, no installs, no complexity. The concept is simple: one website (index.html) where you drop any HTML file and it runs instantly in a sandboxed preview, with tabs so you can have multiple files open at once. When you want to go beyond the browser — like running a crosshair overlay that sits on top of games — a tiny launcher exe handles that. The launcher is a one-time download built automatically by GitHub, sits silently in your system tray, and the website talks to it in the background via a local API. Drop a file, hit "Launch as App", and it opens as a real native window — transparent, always on top, no trace left when you close it. The whole thing is self-contained: the website is one file, the launcher never changes, and nothing requires commands, Node, or any setup ever again.
